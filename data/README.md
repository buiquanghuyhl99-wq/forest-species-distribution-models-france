# Data folder

The raw forest-plot dataset is intentionally excluded from this public repository.

To run the workflow locally, place the private CSV file here:

```text
database_ed.csv
```

The expected default format is:

- separator: semicolon (`;`);
- decimal mark: comma (`,`); 
- binary response variable: `qupe`;
- explanatory variables: beginning in column six.

Do not upload private or restricted forest-inventory data to a public repository.
