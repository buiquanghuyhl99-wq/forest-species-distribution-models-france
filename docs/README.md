# Scientific report

This folder contains the collaborative scientific report for the species distribution modelling project.

The report documents:

- the study context;
- model variables and equations;
- European beech and sessile oak suitability maps;
- comparison between 1961-1990 and 1991-2020;
- ROC curves and performance metrics;
- discussion and forest-management implications.

The report authors are listed in the main repository README.
